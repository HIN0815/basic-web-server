# AWS-WordPress

## Cloud 9 Setup ##

git clone https://github.com/HIN0815/WordPress.git  
cd WordPress/  
chmod +x *.sh  
./setup.sh  

after that you need to run  
source venv/bin/activate  

then use the  
. ./set_config.sh  
command to set the config, not forget the first "."

after that you can run  
./deployment.sh  
to deployment the WordPress
